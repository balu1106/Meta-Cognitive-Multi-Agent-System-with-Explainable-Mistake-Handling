# Meta-Cognitive-Multi-Agent-System-with-Explainable-Mistake-Handling

Exploring multi-agent systems that can detect their own mistakes and explain decisions in a human-understandable way.

---

## Overview

This project explores how multi-agent systems can become more reliable and transparent by adding a layer of self-awareness (meta-cognition) and explainability.

Most existing systems focus only on optimizing outcomes, but they do not reflect on their own mistakes or explain their reasoning clearly. In this project, I try to address this gap by designing agents that can detect errors in their own decisions, learn from them, and communicate those insights in a way that is understandable to humans.

This project is being developed as part of my Bachelor’s thesis (final year project) in Artificial Intelligence and Machine Learning.

---

## Problem

In complex environments, agents interact and make decisions continuously. However, when something goes wrong:

* the system does not clearly identify what failed
* there is no mechanism to learn from that mistake properly
* users cannot understand why a certain decision was made

This reduces trust and makes improvement difficult.

---

## What I am building

* A multi-agent negotiation setup using simulation frameworks
* A meta-cognitive layer that tracks decisions and identifies mistakes
* A simple explainability module that converts agent reasoning into human-readable form
* A small interface to visualize decisions and feedback

---

## Approach

The system is being developed step-by-step:

* Designing agent behaviour and negotiation logic
* Logging interactions and outcomes
* Adding mistake detection rules
* Introducing basic learning/adaptation
* Generating explanations using rule/template-based methods

---

## Tools & Tech

Python, Mesa / NegMAS, Scikit-learn, Streamlit, basic databases (CSV / SQLite)

---

## Expected outcome

The goal is not just performance, but:

* better transparency
* ability to learn from past errors
* improved trust in agent decisions
